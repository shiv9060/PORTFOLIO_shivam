# PORTFOLIO_shivam

Personal portfolio for Shivam Kumar — Software Engineer & Cloud/DevOps Engineer.

Built with **React**, **Vite**, and **Three.js** (react-three-fiber) featuring an interactive 3D city scene.

## Getting Started

### Prerequisites

- Node.js ≥ 18
- npm ≥ 9

### Install dependencies

```bash
npm install
```

### Run in development mode

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

Open [http://localhost:4173](http://localhost:4173) in your browser.

## Tech Stack

- React 18
- Vite 5
- Three.js + @react-three/fiber + @react-three/drei
- CSS (inline styles & index.css)
